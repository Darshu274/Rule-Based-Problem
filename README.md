# Rule-Based-Problem
# Machine Learning Challenge - Winter 2025

This repository contains the solution for the Machine Learning Project Work for the Winter 2025 semester. The project involves two predictive challenges using an individual dataset assigned via Moodle.

---

## 🛠 Project Structure

### Part 1: Regression Task
* **Objective:** Predict the continuous variable `target01` to achieve the best-possible prediction quality.
* **Dataset:** Utilizes `dataset_<ID>.csv` for features and `target_<ID>.csv` for training.
* **Deliverable:** A file named `EVAL_target01_<ID>.csv` containing predictions for the evaluation set.

### Part 2: Rule-Based Logic (Edge Device)
**Objective:** Identify the specific features and simple "if-else" rules used to calculate `target02`.
* **Constraints:** The solution must be implemented within the provided `framework.py` without importing additional packages.
* **Edge Compatibility:** The code must function using basic comparisons and numerical functions for use on an edge device.
* **Deliverable:** The modified file renamed to `framework_<ID>.py`.
